# Malware automated remediation

**Episode description**: In this episode of Defender for Cloud in the Field, Ethan Bremler joins Yuri Diogenes to talk about the new malware automated remediation capability in Microsoft Defender for Storage. Ethan explains the different use cases this capability unblocks and the prerequisites to enable this feature. Ethan demonstrates how to configure this feature and what the overall experience looks like when malware is detected.
