# New secure score

**Episode description**: In this episode of Defender for Cloud in the Field, Yael Genut joins Yuri Diogenes to talk about the new Secure Score in Defender for Storage. Yael explains the new secure score calculation, how to access the new secure score in the Defender Portal, and the new enhancements available in the Posture dashboard.
